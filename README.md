# 🚀 한국 주식 급등 예측 프로그램

AI 딥러닝과 기술적 분석을 활용한 한국 주식 급등 예측 시스템입니다.

## 📋 주요 기능

### 1. 기본 급등 탐지 (`stock_surge_detector.py`)
- 코스피/코스닥 주요 종목 실시간 분석
- 기술적 지표 기반 급등 신호 탐지
- RSI, 이동평균선, 볼린저밴드, 거래량 분석
- 급등 점수 시스템으로 종목 순위 매기기

### 2. AI 딥러닝 예측 (`advanced_stock_predictor.py`)
- LSTM 신경망을 활용한 주가 예측
- 15개 기술적 지표를 활용한 고급 분석
- 미래 주가 예측 및 급등 확률 계산
- 모델 성능 평가 및 정확도 표시

### 3. 웹 애플리케이션 (`streamlit_app.py`)
- 사용자 친화적인 웹 인터페이스
- 실시간 차트 및 시각화
- 개별 종목 상세 분석
- 대화형 대시보드

## 🛠️ 설치 및 실행

### 1. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

```bash
python stock_surge_detector.py
```

### 3. AI 딥러닝 예측 실행
```bash
python advanced_stock_predictor.py
```

```bash
streamlit run streamlit_app.py
```

## 🔍 기술적 지표

### 기본 지표
- **RSI (Relative Strength Index)**: 과매수/과매도 판단
- **이동평균선**: 5일, 20일, 60일 이동평균
- **볼린저 밴드**: 가격 변동성 및 지지/저항선
- **거래량 분석**: 평균 거래량 대비 비율

### 고급 지표 (AI 모델용)
- **MACD**: 추세 전환 신호
- **스토캐스틱**: 모멘텀 지표
- **ATR**: 평균 진폭 범위
- **변동성 지표**: 가격 변동성 측정
- **가격-거래량 관계**: VPT 지표

## 🤖 AI 모델 구조

### LSTM 신경망
```
Layer 1: LSTM(50 units, return_sequences=True)
Layer 2: Dropout(0.2)
Layer 3: LSTM(50 units, return_sequences=True)
Layer 4: Dropout(0.2)
Layer 5: LSTM(50 units)
Layer 6: Dropout(0.2)
Layer 7: Dense(25)
Layer 8: Dense(1)
```

### 특징
- 60일 시퀀스 데이터 학습
- 15개 기술적 지표 활용
- 80/20 훈련/테스트 분할
- Adam 옵티마이저 사용
- MSE 손실 함수

## ⚠️ 투자 유의사항

1. **면책 조항**: 이 프로그램은 정보 제공 목적이며, 투자 조언이 아닙니다.
2. **위험성**: 주식 투자는 원금 손실 위험이 있습니다.
3. **다양한 정보**: 여러 정보를 종합하여 투자 결정을 내리시기 바랍니다.
4. **모델 한계**: AI 예측 모델도 100% 정확하지 않습니다.
