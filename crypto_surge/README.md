# 코인 급등 예측 시스템

주식 급등 예측 시스템을 코인(바이낸스 USDT 마켓)에 적용한 버전입니다.

## 핵심 전략
- 200일선 아래 60일+ 조정 → 최근 45일 이내 돌파 → 현재가 200선 위 0~15% 이내
- 거래량, OBV, 이평선 정배열, RSI, BB, MACD, 펀딩비, 4H 추세 복합 신호

## 주식 버전과의 차이점
| 항목 | 주식 | 코인 |
|------|------|------|
| 데이터 소스 | yfinance | ccxt (바이낸스) |
| 기준선 | 240일선 | 200일선 |
| 최소 조정 | 120일 | 60일 |
| 최대 이격 | 10% | 15% |
| 운영 시간 | 평일 15:40 | 24/7 (4시간 간격) |
| 추가 신호 | 기관/외국인 수급 | 펀딩비, 4H 추세 |

## 설치

```bash
cd crypto_surge
pip install -r requirements.txt
```

## 실행

```bash
# 웹 앱
streamlit run app.py

# 수동 스캔
python run_scan.py

# 자동 스케줄러 (4시간 간격)
python scheduler.py
```

## 환경변수 설정
`.env.example`을 `.env`로 복사 후 텔레그램 설정 입력
